Ruby

After finishing basic SQL, I did some more research to see what other

programming languages were recommended. I found that Ruby was another

highly recommended language to know. Ruby is like a mixture of SQL, HTML,

and JavaScript. It is a middle ground to all of these, it is not meant

to make complex games, but it is not meant to be solely databases. There

are some other things that Ruby can do that SQL and JavaScript cant.
