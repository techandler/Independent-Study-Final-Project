SQL

After learning a lot about how to make a game, and organize a page to

display the game and make it playable, I did some research to find out

other good coding languages to know. I found that knowing SQL (Structured

Query Language) is highly recommended by many sites. Unlike JavaScript

and Python, SQL does not make applications that you can play. SQL is more

of a management tool that can be used to organize databases.
