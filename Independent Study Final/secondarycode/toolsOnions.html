Tools

For my coding with python, I only used JetBrains PyCharm. I think that it worked
well, allowing me to easily edit my code as well as giving me the ability to
test my code directly from the editor. It would then show in the console if
there were any issues, and where they were in the code.

For JavaScript, I tried Code Writer, and Atom. Both of these applications had
their advantages. Code Writer was a much more simple looking application, that
the ability to tab multiple lines, and auto tab. Atom had a much more unique look
that also had many useful tools. Overall, I enjoyed using Atom more, it allowed
for a customizable look without losing many functions.

For HTML and CSS, I tried the Code Writer, Brackets, and Atom. Overall, I
would say that Atom was my favorite editor. It had great functionality along
with an ability to change themes for the editor and syntax. It also had the
capability to change basically everything, including making a startup script
which would run whenever you opened something.
