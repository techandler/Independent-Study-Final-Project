HTML

After I learned plenty of python and javascript, I decided that I needed to

learn a better way to implement these into systems that make them easier

to get to, and easier to use. I decided that it would be best for me to

start learning HTML (HyperText Markup Language). This would not allow me to

put my python applications directly into a browser, but it would allow me to

neatly insert my javascript applications, so they can be played on a web browser

like Google Chrome. Everything that you are seeing on this screen is based from

HTML, it is the foundation of all of the pages.
