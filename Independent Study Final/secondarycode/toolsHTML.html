Tools Used

Throughout all of my coding, I used multiple different tools to learn

the different languages, and I used several different programs to write

my code. To learn the basics of the languages, I used Codecademy which

is a website whose goal is to teach coding languages to as many people

as possible. After learning the basics of the different coding languages,

for some of them, I had to do additional learning in order to complete

what I wanted to. I would often do this by watching a youtube video or by

using w3schools.com. Once I had learned a coding language, I often practiced

it using a code writing application. I tried out different applications in

order to see which I liked the most, and which one felt the best to use.
